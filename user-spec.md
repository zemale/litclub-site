# User Spec: ЛитКлуб — интерактивный учебник по литературе

## Что делаем

Сайт-учебник по школьной литературе (5-11 класс) с геймификацией:
- 50+ произведений из базовой школьной программы
- Краткие содержания, биографии авторов, примеры эссе
- **Дискуссии** — чат с виртуальным литературоведом через Gemini API
- **Визуальные новеллы-квесты** по сюжетам произведений
- XP, достижения, стрик в LocalStorage

## Целевая аудитория

Школьники 5-11 классов, их родители, учителя литературы.

## Почему это важно

Сейчас ученики заучивают сочинения или покупают готовые. ЛитКлуб делает изучение литературы интерактивным — через погружение в сюжет (квесты) и живые дискуссии, а не зубрёжку.

## Ключевые сценарии

1. Ученик 8 класса готовится к контрольной по «Герою нашего времени» → проходит квест, обсуждает образ Печорина с ботом → пишет своё эссе
2. Учитель литературы даёт ссылку на краткое содержание → ученик читает за 5 минут, получает XP
3. Ученик проходит диагностику → сайт рекомендует произведения по уровню
